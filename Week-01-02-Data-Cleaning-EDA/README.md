# Week 1 & 2: Data Cleaning & Exploratory Data Analysis (EDA)

## Overview

During the first two weeks of the AnalystLab Africa Data Science Internship, I focused on data cleaning and exploratory data analysis (EDA) using the Titanic and Housing Price datasets. The objective was to prepare the datasets for analysis by improving data quality and exploring patterns that could support future statistical analysis and machine learning tasks.

## Objectives

* Inspect dataset structure and data types.
* Identify and handle missing values.
* Detect and remove duplicate records.
* Explore numerical and categorical variables.
* Perform univariate, bivariate, and multivariate analysis.
* Visualize data distributions and relationships.
* Summarize key insights from the analysis.

## Datasets

* **Titanic Dataset** – Passenger demographics and survival information.
* **Housing Price Dataset** – Residential property features and sale prices.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Learning Outcomes

* Applied data cleaning techniques to improve dataset quality.
* Performed exploratory data analysis to understand data distributions and relationships.
* Identified patterns, trends, and potential predictors within both datasets.
* Strengthened practical skills in Python and data visualization.
* Reinforced the importance of preparing data before statistical analysis and machine learning.

## Conclusion

Data cleaning and exploratory data analysis form the foundation of every successful data science project. By ensuring data quality and understanding the characteristics of each dataset, meaningful insights can be generated and more reliable analytical models can be developed in subsequent stages of the project.
